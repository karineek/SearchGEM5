Section{Background}

\gem5{} is a state of the art
discrete time simulator for  logic circuits.
It is often used to try out the logic design of new electronic
components
such as memory cache systems, FPGAs, even CPUs.
\gem5{} is a large open source project hosted on GitHub
written mostly in C++ and Python.
For the SSBSE~2023 Challange Track,
Dr.~Bobby~R.\ Bruce
prepared a dedicated GitHub clone%
\footnote{%
7 June 2023
\URL{https://github.com/BobbyRBruce/gem5-ssbse-challenge-2023}
}.
The 2023 challange track version
is based on \gem5{} staging branch v23.0
but with the latest \gem5{} features and improvements
merged into a stable release
and the addition of  ssbse-challenge-examples.
